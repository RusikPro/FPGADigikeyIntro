# DigiKey FPGA Tutorial Projects

Verilog implementations of various FPGA designs from DigiKey tutorials.
Remake on DigiKey tutorial: improved Verilog code, made-up assignments.

**Based on:** https://www.youtube.com/watch?v=lLg1AgA2Xoo&list=PLEBQazB0HUyT1WmMONxRZn9NmQ_9CIKhb
**Original Repository:** https://github.com/ShawnHymel/introduction-to-fpga

## Hardware
- FPGA: iCE40 (Lattice)
- Clock: 12 MHz
- Tools: apio

## Building

```bash
cd \<path/to/project\>
apio build
apio upload
```
